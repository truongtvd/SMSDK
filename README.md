## Installation

SMSDK is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
use_frameworks!
pod 'SMSDK', :git => 'https://github.com/truongtvd/SMSDK.git'
```
